Política de Privacidade – World Rush Run

Última atualização: 01/02/2026

O aplicativo World Rush Run respeita a privacidade dos usuários. Esta Política de Privacidade descreve como as informações são coletadas, usadas e compartilhadas ao utilizar o aplicativo.

1. Informações coletadas

O aplicativo não coleta informações pessoais diretamente, como nome, email ou telefone.

No entanto, serviços de terceiros utilizados no app (como plataformas de anúncios) podem coletar automaticamente certos dados, incluindo:

  • Identificadores do dispositivo
  • Endereço IP
  • Dados aproximados de localização
  • Informações de uso do aplicativo
  • Dados de diagnóstico e desempenho

Esses dados são utilizados principalmente para exibição de anúncios, análise de desempenho e melhoria da experiência do usuário.

2. Anúncios

O World Rush Run exibe anúncios, incluindo anúncios recompensados (por exemplo, para ganhar vidas, abrir baús diários, continuar jogando ou dobrar pontos).

Os provedores de anúncios podem usar cookies ou tecnologias semelhantes para mostrar anúncios relevantes.

Política de privacidade do Google:
https://policies.google.com/privacy

3. Crianças

Este aplicativo não é direcionado a crianças menores de 13 anos. Não coletamos intencionalmente dados pessoais de crianças.

4. Compartilhamento de dados

Não vendemos nem compartilhamos dados pessoais diretamente. Dados coletados automaticamente podem ser processados por serviços de terceiros apenas para fins de anúncios e análise.

5. Segurança

Adotamos medidas razoáveis para proteger as informações, mas nenhum método de transmissão ou armazenamento eletrônico é 100% seguro.

6. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos que o usuário revise esta página ocasionalmente.
